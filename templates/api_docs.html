API Documentation

Custom API Endpoint

Our custom API endpoint allows you to fetch pincode details programmatically. Below is the documentation for using the API.

Endpoint

GET /api/pincode/<pincode>

Parameters

Parameter Description
pincode The Indian postal code (pincode) you want to look up. Must be a 6-digit number.

Response

The API returns a JSON response with the following structure:


{
    "status": "success",
    "pincode": "110001",
    "details": [
        {
            "Name": "Connaught Place",
            "BranchType": "Head Post Office",
            "District": "New Delhi",
            "State": "Delhi",
            "Country": "India"
        }
    ]
}

Example Request

To fetch details for pincode 110001, make a GET request to:

GET /api/pincode/110001

Example Response


{
    "status": "success",
    "pincode": "110001",
    "details": [
        {
            "Name": "Connaught Place",
            "BranchType": "Head Post Office",
            "District": "New Delhi",
            "State": "Delhi",
            "Country": "India"
        }
    ]
}

Error Responses

If the pincode is invalid or no details are found, the API will return an error response:


{
    "status": "error",
    "message": "No details found for this pincode."
}

Rate Limiting

Currently, there are no rate limits for the API. However, please use it responsibly.
